# CodeAlpha_DiseasePrediction

## Project Description
This project predicts whether a breast cancer tumor is benign or malignant using a Machine Learning model built with Scikit-learn.

## Features
- Uses the Breast Cancer dataset from Scikit-learn
- Splits the dataset into training and testing sets
- Trains a Random Forest Classifier
- Predicts the test data
- Calculates and displays the model accuracy

## Technologies Used
- Python
- Scikit-learn

## Output
Disease Prediction Model

Accuracy: 0.9649 (approximately 96.49%)# CodeAlpha_DiseasePrediction
